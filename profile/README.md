## About

Sugra Systems, Inc. operates an intelligence infrastructure platform. The stack aggregates sovereign, intergovernmental, and academic data into a single normalized API, exposes it to AI agents through a Model Context Protocol gateway, and surfaces it in an end-user application.

The platform is domain-agnostic. Financial markets are the first vertical, not the identity.

## Platform

`sugra.ai` is the flagship platform. Three products run on it:

| Product | Surface | Status |
|---|---|---|
| **Sugra API** | [sugra.ai](https://sugra.ai) | Live |
| **Sugra Assist** | Conversational interface for natural-language queries on platform data | Planned |
| **Sugra App** | [app.sugra.ai](https://app.sugra.ai) | Early development |

## Data Sources

The catalog is organized in tiers. Public surfaces enumerate Tier S (sovereign, regulator, central bank), Tier I (intergovernmental), and Tier A (academic and non-profit foundation) sources individually. Commercial upstreams are wrapped under Sugra-branded products: Sugra Finance, Sugra News, Sugra Crypto, Sugra Forex.

Live catalog: [sugra.systems/api/data-providers](https://sugra.systems/api/data-providers).
